Real-Time Monitoring

MAX30102 (HR/SpO2) + AD8232 (ECG) sensors. ESP32 publishes MQTT every 5 s → AWS IoT Core → Lambda → DynamoDB PatientData table.

Multi-Layer Security

AES-256-CBC payload encryption, HMAC-SHA256 digital signatures, consortium blockchain integrity. All verified server-side by Lambda before DynamoDB write.

AWS DynamoDB Storage

PatientData table stores all ESP32 records. SecurityLogs table tracks anomalies. Both queried by Lambda at GET /dashboard and returned to this S3-hosted frontend.
